\# MCU Data Logger



A modular MCU-based Data Logger designed in KiCad as part of a PCB Design Internship.



\## Project Overview



This project demonstrates the complete hardware design workflow for an ATmega328P-based data logger using hierarchical schematics and modular PCB design practices.



The design includes:

\- ATmega328P Microcontroller

\- DS1337 Real Time Clock (RTC)

\- 24LC1025 EEPROM

\- I2C Interface

\- UART Interface

\- ICSP Programming Header

\- GPIO Expansion Header

\- Power Supply Section
